# CRUDs em C

Sistemas de gerenciamento desenvolvidos em linguagem C durante o curso de Lógica de Programação no SENAI-BA.

Cada projeto aplica os mesmos conceitos base: structs, enums, arrays estáticos, validação de entrada e menu interativo com `do-while`.

## Projetos

| # | Sistema | Destaques |
|---|---------|-----------|
| 01 | Index | Navegação central entre os CRUDs |
| 02 | Biblioteca | Cadastro de livros com filtro por status |
| 03 | Contatos | Agenda com busca por nome |
| 04 | Alunos | Cadastro com cálculo de média e situação |
| 05 | Produtos | Controle de estoque com filtro por categoria |
| 06 | Usuários | Cadastro com perfis de acesso |
| 07 | Playlist | Gerenciador de músicas com formatação de duração |
| 08 | Veículos | Catálogo com filtro por tipo |
| 09 | Receitas | Busca por ingrediente |
| 10 | Filmes | Catálogo com busca por título e filtro por gênero |

## Compilar e executar

```bash
gcc nome-do-arquivo.c -o programa
./programa
```
